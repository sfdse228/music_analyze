# 🎵 Анализатор статистики прослушиваний

Программа для анализа истории прослушиваний музыки из JSON-файла с визуализацией данных.

## 📋 Описание

Программа загружает данные о треках из JSON-файла и выполняет анализ:
- Суммарное время прослушивания
- Топ-жанры
- Самый прослушиваемый трек
- Топ-исполнители
- Визуализация в виде графиков

## 📁 Структура проекта
music-analyzer/
├── music.json # Данные с треками
├── music_analyzer.py # Главный скрипт
├── requirements.txt # Зависимости
├── stats_report.txt # Сгенерированный отчёт
└── README.md # Описание проекта
